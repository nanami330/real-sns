## 環境構築方法
### インストール
1.リポジトリをクローン
- `git clone`

2.必要なパッケージをインストール
- `npm install`

### ローカル環境の立ち上げ
1.バックエンドの起動
- `cd real-sns/backend`
- `npm start`

2.フロントエンドの起動
- `cd real-sns/frontend`
- `npm start`

下記のローカル環境にアクセスできれば　OK <br>
localhost:3000 <br>

## 技術スタック
データベース: MongoDB

バックエンド: Express, Node.js

フロントエンド: React
